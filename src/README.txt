Skedi Business Scheduler
The purpose of this application is to help businesses with large userbases manage and schedule appointments.

Author: Alberto Sosa
Contact Info: asosa8@wgu.edu
Version: 5.30.98b
Date: 1/25/2022
IDE: IntelliJ IDEA Community Edition 2021.2.3
JDK: Java SE 11.0.12
JavaFX: JavaFX-SDK-11.0.2
MySQL Connector Driver: mysql-connector-java-8.0.27

Directions:
Launching application will prompt you with a login form. 
Once successfully logged in, you are presented with the main menu.
Under the appointment view, there are buttons to filter, add, update, or delete appointments.
Under the customer view, there are buttons to add, update, or delete customers. 
At the bottom right there are buttons to generate the following reports: 
	- Appointment Type Report calculates the amount of appointments that are in the database of a given month and type.
	- Contact Schedule Report generates a schedule for all appointments of a specified contact.
	- Appointment Customer Report calculates the amount of appointments that a given customer has.
There is also a logout button at the bottom right which returns to the login screen. 

Additional report information:
I chose to implement a report that displays the amount of appointments a selected customer has.
